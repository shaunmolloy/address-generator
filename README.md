# Address generator

Address generator is a node module that outputs a random address.

## Installation

Install the package with either npm:

```sh
npm i @shaunmolloy/address-generator
```

Or with yarn:

```sh
yarn @shaunmolloy/address-generator
```

## Usage

```sh
address-generator
```

Example of output:

```
street: 79 Chorley Road
town: Bolton
county: Lancashire
postcode: BL7 0GJ
```
